## Euclidea原题
两条轴互相垂直，交点为O。A为平面上不在轴上的一点。仅用直尺，作OA关于任意一条轴的对称图形。

## 解法
在y轴OA同侧取一点B，BA的延长线交x轴于C。
在y轴OA异侧取一点D，DC的延长线交OA于E。
延长BE，交x轴于F，连接DF，交BA的延长线于G。
连接OG，OG与OA关于
